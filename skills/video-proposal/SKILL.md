# Video Proposal

Analyze indexed video collections and propose short-form video concepts with curated clip selections. Works with any collection — proposals are derived from the actual metadata, not hardcoded.

## Prerequisites

- A `collection-index.json` file generated by the `vidmeta` skill (contains per-clip metadata: tags, mood, motion, descriptions, audio info)
- The `capcut-api` MCP server running on localhost:9001 (for building drafts)
- CapCut desktop app installed (for draft import)

## Workflow

### Phase 1: Analyze Collection

Run the analysis script to categorize all clips and generate proposals:

```bash
python scripts/analyze_and_propose.py <path-to-collection-index.json> --output <output-dir>/video-proposals.json --name "My Trip Name"
```

The `--name` flag sets the collection name used in proposal titles. If omitted, it's derived from the metadata locations or folder name.

This produces a `video-proposals.json` containing:
- **collection_summary** — tone, style, themes, locations, dominant mood (all derived from metadata)
- **clip_categories** — clips sorted into detected categories (only categories with clips appear)
- **best_clips** — hero shots, opening hooks, closing shots, emotional moments
- **video_proposals** — one highlight reel + one proposal per category that has 3+ clips

#### Categories

The script recognizes these content categories from clip tags:

| Category | Example tags |
|----------|-------------|
| `scenic_landscape` | mountains, viewpoint, sunset, valley, lake |
| `driving_road` | road, car, motorcycle, highway, bridge |
| `family_people` | child, family, friends, portrait, celebration |
| `cultural_village` | village, temple, market, heritage, architecture |
| `nature_flora` | forest, flowers, garden, autumn, bloom |
| `atmospheric_mood` | fog, mist, rain, night, golden_hour |
| `food_lifestyle` | food, restaurant, cafe, cooking, street_food |
| `camping_outdoor` | camping, hiking, trail, climbing, kayak |
| `water_scenes` | ocean, beach, river, waterfall, waves |
| `urban_city` | city, skyline, metro, neon, nightlife |

Categories without enough clips (< 3) are skipped for proposals but still appear in the categorization output. Clips not matching any category go to `uncategorized`.

### Phase 2: Research Trending Styles (Optional)

To enhance proposals with current platform trends, spawn parallel research agents:

1. **Trend research** — search for trending TikTok/Reels video formats, transitions, music, and pacing for the collection's themes. Write to `references/trending-styles.md`
2. **Editing techniques** — search for editing best practices relevant to the collection's dominant content (e.g., landscape color grading, food close-ups, urban pacing). Write to `references/editing-techniques.md`

Use the Task tool with `trend-researcher` subagent type for both searches. Reference files in `references/` contain cached results from prior research runs.

### Phase 3: Build Video Draft

To execute a proposal using the capcut-api MCP tools:

1. Load the proposal from `video-proposals.json`
2. For each proposed video:
   - Call `mcp__capcut-api__create_draft` (1080x1920 for vertical)
   - Call `mcp__capcut-api__add_video` for each clip in the proposal's `suggested_clips` list
   - Call `mcp__capcut-api__add_text` for title/end card overlays
   - Call `mcp__capcut-api__save_draft`
3. Install the draft into CapCut using `scripts/install_to_capcut.py`

#### VectCut API Constraints (learned from usage)

- **Transition names** are capitalized with underscores: `Dissolve`, `Black_Fade`, `Wipe_Right` (not `fade_in`)
- **Text animations** use `Fade_In`, `Fade_Out`, `Zoom_In` etc. (capitalized)
- **Text segments cannot overlap** in time on the same track — combine multi-line text into one element using `\n`
- **Video paths**: the server copies local files to its working dir as `.mp4` assets. After `save_draft`, run `install_to_capcut.py` to copy assets to CapCut and patch paths

#### install_to_capcut.py

After saving a VectCut draft, install it into CapCut:

```bash
python scripts/install_to_capcut.py <draft_id> <draft_name>
```

### Phase 4: Track Progress

Update `video-proposals.json` to track which proposals have been built:

```json
{
  "video_proposals": [
    {
      "id": "my-trip-highlight",
      "status": "completed",
      "draft_id": "dfd_cat_...",
      "capcut_name": "my-trip-highlight",
      "completed_at": "2026-02-09T..."
    }
  ]
}
```

Status values: `proposed` | `in_progress` | `completed` | `skipped`

## Scripts

| Script | Purpose |
|--------|---------|
| `scripts/analyze_and_propose.py` | Analyze collection metadata, categorize clips, generate video proposals |
| `scripts/install_to_capcut.py` | Copy VectCut draft into CapCut with path fixes |

## Video Concept Templates

When proposing videos, consider these proven formats:

| Format | Duration | Clips | Style |
|--------|----------|-------|-------|
| Trip highlight reel | 15-30s | 5-7 | Fast montage, dissolve transitions |
| Cinematic mood piece | 30-60s | 8-12 | Slow, atmospheric, speed ramps |
| POV driving montage | 15-30s | 4-6 | Dashcam compilation, speed ramps |
| People & moments story | 30-60s | 8-10 | Mix of scenic + personal moments |
| Nature showcase | 15-30s | 4-6 | Gentle pacing, soft transitions |
| Adventure reel | 15-30s | 4-6 | Quick cuts, energetic |
| Food & lifestyle reel | 15-30s | 4-6 | Close-ups, warm grading |
| Urban energy montage | 15-30s | 4-6 | Beat-synced, fast cuts |
